# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-02-19
### Added
- **Kiosk Mode:** Secure PIN entry (Default: 1234) with auto-lock.
- **Room Dashboard:** Horizontal scrolling carousel for room navigation.
- **One-Tap Attendance:** Instant check-in/out with local persistence (Dexie.js).
- **Absence Alerts:** Visual 'LATE' badge and prompt for children not checked in by their cutoff time.
- **Offline First:** Full functionality without internet connection.
